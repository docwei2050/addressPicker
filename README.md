# addressPicker
 商城收货地址选择
 1.顶部导航栏不是系统的TabLayout，因为系统提供的宽度不能跟着内容宽度变化
 2.指示器的宽度和X方向的动画思路借鉴其他库，，就一个方法
 3.只做了3级联动，4级地址联动数据因为经常变化就不做了
